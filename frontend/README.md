# AI Banking System

The AI Banking System is a modern web-based banking application that integrates React and Spring Boot with AI modules to deliver secure, intelligent, and user-friendly digital financial services.

## Highlights
- Secure authentication with JWT and role-based access
- Account management, fund transfers, transaction history
- Multiple accounts support with unique account numbers and IFSC codes
- AI modules for fraud detection and loan approval assistance
- Responsive React UI with protected routes and blue theme

## Architecture
- Frontend: React (Vite), component-based UI, axios client with interceptors
- Backend: Spring Boot REST APIs, H2/MySQL-ready, Spring Security + JWT
- Database: Relational schema for users, accounts, and transactions
- AI: Modules for fraud detection patterns and loan decision support
